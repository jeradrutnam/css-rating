# CSS Rating Component

### Basic usage

Example:

```html
<link href="css/rating.css" rel="stylesheet">
```

```html
<div class="rating">
    <i class="icon fw fw-star"></i>
    <i class="icon fw fw-star"></i>
    <i class="icon fw fw-star"></i>
    <i class="icon fw fw-star"></i>
    <i class="icon fw fw-star"></i>
</div>
```
